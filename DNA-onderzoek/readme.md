## Korte beschrijving van de cursus
Onze erfelijke eigenschappen liggen opgeslagen in ons DNA. DNA bestaat uit een soort letters die samen een genetische code vormen. Deze genetische code is voor iedereen een beetje verschillend. Om te kunnen bepalen welke genetische variant je hebt voor een bepaalde eigenschap, kun je je DNA (laten) testen. De eigenschap die we gaan onderzoeken is het wel of niet kunnen proeven van een bittere stof (PTC). Je leert hierbij meer over DNA, welke gevolgen kleine verschillen in de genetische code kunnen hebben en wat je allemaal te weten kunt komen met DNA-onderzoek.
